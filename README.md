Exercício criado para a prática de Web API, utilizando fetch e Local Storage.

O objetivo central é permitir que um usuário preencha um formulário de endereço no navegador e que esses dados sejam manipulados dinamicamente usando JavaScript. A implementação faz uma chamada a uma API para buscar dados de endereço com base no CEP e armazena ou recupera dados do Local Storage para persistência simples no cliente. O projeto é composto por arquivos HTML, JavaScript e CSS, e serve como prática para integrar lógica de API no navegador e gerenciar estado local do aplicativo.
